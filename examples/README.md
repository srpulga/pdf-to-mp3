# Examples

This folder is for example PDF files that you want to convert to MP3.

## Usage

1. Place your PDF files here
2. Run the application: `streamlit run src/app.py`
3. Upload the PDF from this folder
4. Convert to MP3

## Notes

- This folder is not tracked by git (to avoid large files in the repository)
- You can add any PDF file here for testing
- Generated MP3 files will be saved in the `output/` folder
